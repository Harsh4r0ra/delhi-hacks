React + Vite + Tailwind CSS + ShadCN UI frontend for the ByzantineMind Byzantine Fault Tolerant AI Consensus Engine.

## Setup

```bash
cd frontend
npm install
npm run dev
```

Runs on `http://localhost:8080`. Expects the FastAPI backend on `http://localhost:8000`.

## Stack

- **React 18** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS** + **ShadCN/UI** (components)
- **Framer Motion** (animations)
- **TanStack Query** (data fetching)
- **Recharts** (data visualization)
